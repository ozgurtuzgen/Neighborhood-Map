# Neighborhood-Map

This is an application that shows the places around.

## Getting Started

- Just clone the code.
- Open the index.html in the browser that you want.

### Prerequisites

- JQuery, Knockout, Bootstrap CDNs are used. So you need internet available.

### Installing

- Nothing to install.
